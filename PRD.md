# Image Background Remover - MVP 需求文档

**版本：** v0.1  
**日期：** 2026-03-16  
**作者：** Zane  

---

## 1. 产品概述

一个在线图片背景去除工具，用户上传图片后自动去除背景，支持下载透明背景结果图。

**目标用户：** 电商卖家、设计师、普通用户  
**核心价值：** 免费、快速、无需注册、隐私安全（图片不落盘）

---

## 2. 技术架构

| 层级 | 技术选型 |
|------|----------|
| 前端 | Next.js + Tailwind CSS + shadcn/ui |
| 部署 | Cloudflare Pages |
| API 代理 | Cloudflare Workers |
| 背景去除引擎 | Remove.bg API |
| 存储 | 无（纯内存处理） |

**数据流：**
```
用户上传图片 → Cloudflare Worker（隐藏 API Key）→ Remove.bg API → 返回结果图 → 用户下载
```

---

## 3. MVP 功能范围

### 3.1 核心功能（Must Have）

#### F1 - 图片上传
- 支持点击上传和拖拽上传
- 支持格式：JPG、PNG、WebP
- 单文件大小限制：≤ 10MB（Remove.bg 免费版限制）
- 上传后立即触发处理，无需额外点击

#### F2 - 背景去除处理
- 调用 Cloudflare Worker 代理 Remove.bg API
- 处理中显示 loading 状态
- 处理失败显示友好错误提示（如：超出每日免费额度）

#### F3 - 结果预览
- 左右对比展示：原图 vs 去背景结果
- 结果图使用棋盘格背景展示透明区域

#### F4 - 下载结果
- 下载 PNG 格式（保留透明通道）
- 文件名：`原文件名_removed.png`

#### F5 - 重新上传
- 处理完成后提供"再处理一张"按钮
- 清空当前状态，回到初始上传界面

### 3.2 非功能需求

- **响应速度：** 页面首屏加载 < 2s（Cloudflare CDN 加速）
- **处理时长：** 取决于 Remove.bg API，通常 2-5s，需有超时提示（30s）
- **移动端适配：** 响应式布局，支持手机浏览器使用
- **隐私：** 图片仅在内存中流转，不写入任何存储

### 3.3 不在 MVP 范围内（Out of Scope）

- 用户注册 / 登录
- 批量处理
- 自定义背景替换
- 历史记录
- 付费订阅

## 5. Cloudflare Worker API 设计

### POST `/api/remove-bg`

**Request:**
```
Content-Type: multipart/form-data
Body: { image: File }
```

**Response (成功):**
```
Content-Type: image/png
Body: <PNG binary>
```

**Response (失败):**
```json
{ "error": "错误描述", "code": "ERROR_CODE" }
```

**错误码：**
| Code | 含义 |
|------|------|
| `FILE_TOO_LARGE` | 文件超过 10MB |
| `UNSUPPORTED_FORMAT` | 不支持的格式 |
| `API_QUOTA_EXCEEDED` | Remove.bg 额度耗尽 |
| `TIMEOUT` | 处理超时 |
| `UNKNOWN` | 未知错误 |

## 7. 验收标准

| 功能 | 验收条件 |
|------|----------|
| 上传 | 支持拖拽和点击，格式/大小校验有提示 |
| 处理 | loading 状态正常，成功率与 Remove.bg 一致 |
| 预览 | 左右对比清晰，透明区域棋盘格展示 |
| 下载 | PNG 文件透明通道完整 |
| 移动端 | iPhone Safari / Android Chrome 可正常使用 |
| 错误处理 | 各类错误有友好提示，不显示原始报错 |
